# SET UP
- install `FVM` (Flutter Version Management) or install `SDK`
- using flutter `3.0.5`
- run `flutter doctor` and do follow the suggest step for make the check on doctor summary
- run `flutter pub get`

# STEP FOR RUN
- open Simulator
- run `flutter run`
